## 🚀 Quick Start (3 Steps)

### 1. Install Dependencies
```bash
python -m venv .venv
pip install -r requirements.txt
```

### 2. Start Ollama (separate terminal)
```bash
ollama serve
ollama pull mistral
```

### 3. Run the System
```bash
# Choose one:
python rag_agent.py              # Interactive chat
python main.py                   # REST API server
streamlit run streamlit_app.py   # Web dashboard
```
